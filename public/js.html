Calculations

Calculation functionality is accessed through the calc object.

Confidence Interval


calc.confidenceInterval(conLev,ss,pop,perc);
param type value default description
conLev percent (%) 95 || 99 null Confidence level
ss number >= 0 null Sample Size
pop number >= 0 null Population
perc percent (%) 0 - 99 null Percentage

NPS Score


calc.npsScore(promoters,detractors,total);
param type value default description
promoters percent (%) || count 0 - 100 || >= 0 null Promoters
detractors percent (%) || number 0 - 100 || >= 0 null Detractors
total number >= 0 null Total Population
