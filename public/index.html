Usage

You will need to include the virgil.min.css and virgil.min.js files in your page.

Example:


    <!DOCTYPE html>
    <html>
      <head>
        ...
        <link href="assets/css/virgil.min.css" rel="stylesheet" />
      </head>
      <body>
        ...
        <script src="assets/js/virgil.min.js"></script>
      </body>
    </html>
